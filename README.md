# tiptap-vue

用tiptap插件的形式 + vue，仿语雀，Notion，做了一个编辑器,不是完成品，有些功能还在调研中。

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
